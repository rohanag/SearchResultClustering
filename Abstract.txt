Title : Web Search Results Clustering
Project Type: Software

Abstract:

Description: User enters a search term, and k-means++ is used to cluster the top 20 results returned by Bing. User is presented with different clusters and 3 labels that describe each cluster. If user wants to obtain more results about a cluster, he/she can click on the cluster description label.

Data-set : 20 search results taken from Bing API based on search term provided by the user. 

Evaluation Method: Purity and Cluster labels are evaluated subjectively.

Results: Average purity over 3 test cases is 88.33%, and the clusters generated by the algorithm are successful in identifying the topic of the cluster.
